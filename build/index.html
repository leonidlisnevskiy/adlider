CPA-трафик

  • выгодная модель оплаты

  • высокая эффективность

  • отстутствие рисков

  • баннерный формат

  • direct-link формат

CPA в переводе с английского означает «оплата за действие», т.е. за совершение посетителем сайта полезных, нужных действий. Например, такими действиями могут быть: регистрация, подписка на рассылку, заполнение анкеты, скачивание файла и т.п.

CPI-трафик

  • эффективное расходование
    средств

  • результативность

  • баннерный формат

  • direct-link формат

  • search формат

Это вариант сотрудничества, который подразумевает оплату за установку (инсталл) приложения. Владелец сайта размещает на нем рекламу, предоставленную партнерской программой, и получает определенную сумму вознаграждения за каждый факт установки.

Видеореклама

  • высокая кликабельность

  • невысокая цена

  • autoplay формат

  • pre-roll формат

  • overlay формат

Это реклама, которая воспроизводит показ видеорекламы с уже встроенной последующей трансляцией видеоконтента. Имеет адаптивный размер с последующей трансляцией рекламного ролика, со встроенным плеером. Запуск рекламного видео ролика происходит автоматически без участия пользователя.

Тизерный трафик

  • большая аудитория

  • широкая настройка

  • простота изготовления

  • дешевезна

Это рекламное сообщение, построенное как загадка, которое содержит часть информации о продукте, но при этом сам товар не демонстрируется. Тизеры обычно появляются на раннем этапе продвижения товара и служат для создания интриги вокруг него.
